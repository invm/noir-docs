🛠️ Installation

Download the latest release from here for your platform.

MacOS​

MacOS users will choose between arm-based and intel-based processors. In the files above the arm-based processor files are ending with _aarch64.dmg and the intel-based processor files are ending with _x64.dmg. After

The app is not signed for mac, thus requiring to change app attributes after installation by running this command from the terminal:

xattr -cr ./Noir.app

Linux​

Linux users can choose between the .deb package and the AppImage.

Windows​

Windows users can choose between the .exe installer and the .msi installer.
